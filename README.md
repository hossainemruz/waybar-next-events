# waybar-next-events

Show upcoming calendar events in waybar.

## Build

```bash
mise run build
```

## Format

```bash
mise run format
```

## Lint

```bash
mise run lint
```

## Test

```bash
mise run test
```

## Run all checks

This will run `build`, `format`, `lint`, and `test` in parallel.

```bash
mise run ci
```

## Run

```bash
go run ./cmd/waybar-next-events
```
